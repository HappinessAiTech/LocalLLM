LocallLLM 개인정보 처리방침 (Privacy Policy)

전승룡(Jeon SeungRyong)(이하 '개발자')은(는) LocallLLM 애플리케이션(이하 '앱')을 이용하는 사용자의 개인정보 보호를 매우 중요하게 생각하며, 관련 법령을 준수하고 있습니다. 본 개인정보 처리방침은 앱 내에서 수집, 사용, 처리되는 정보의 종류와 방식에 대해 설명합니다.

1. 개인정보의 수집 및 이용 목적
LocallLLM은 오프라인 기반의 로컬 인공지능(AI) 어시스턴트 앱으로 설계되었습니다. 사용자의 프라이버시를 극대화하기 위해 대부분의 데이터는 기기 내부에서만 처리되며, 외부 서버로 전송되지 않습니다.

대화 기록 및 텍스트 데이터: 사용자가 입력한 프롬프트, 채팅 내역, 첨부된 문서(PDF, DOCX 등)의 텍스트는 전적으로 사용자 기기의 내부 저장소(로컬 데이터베이스)에만 저장되며 훈련 목적으로 수집되거나 개발자에게 전송되지 않습니다.

파일 및 폴더 접근 (AI 작업 폴더): 사용자가 직접 선택하고 권한을 부여한 디렉토리(Work Folder)에 한해 텍스트 파일을 읽고 쓰는 기능을 수행합니다. 이 과정에서 수집된 파일 데이터는 기기 내부의 AI 모델 처리용으로만 활용됩니다.

기기 제어 및 상태 정보 (카메라, 배터리 등): * 카메라 권한: 기기의 손전등(Flashlight) 켜기/끄기 제어 목적으로만 사용되며, 사진이나 영상을 촬영, 저장, 전송하지 않습니다.

설치된 앱 목록, 배터리 상태, 기기 정보: 사용자의 음성/텍스트 명령에 따라 앱을 실행하거나 상태를 확인하기 위해 기기 내부에서만 로컬로 조회 및 활용됩니다.

2. 외부 서비스 제공을 위한 정보 전송
앱의 특정 기능을 사용하기 위해 아래와 같이 일부 정보가 외부 서비스로 전송될 수 있습니다.

웹 검색 기능 (선택 사항): 사용자가 앱 설정에서 '웹 검색(Web Search)' 기능을 활성화한 경우, 사용자의 질문(검색어)이 검색 결과를 제공하기 위해 선택한 검색 엔진(Tavily, Serper(Google), DuckDuckGo 등)으로 전송됩니다.

AI 모델 다운로드: 초기 구동 시 또는 설정에서 AI 모델(Gemma 4 등)을 다운로드할 때, 사용자의 IP 주소가 모델이 호스팅된 서버(예: HuggingFace)에 노출될 수 있습니다.

3. 광고 네트워크 (제3자 정보 제공)
앱은 사용자에게 무료 서비스를 제공하기 위해 Google AdMob을 통한 배너 광고를 포함하고 있습니다.

Google AdMob은 광고 제공 및 맞춤형 광고 송출을 위해 기기 식별자(Advertising ID), IP 주소, 앱 사용 기록 등의 비식별 데이터를 수집할 수 있습니다.

관련된 자세한 정보는 Google의 개인정보 처리방침에서 확인할 수 있습니다.

4. 개인정보의 보관 및 파기
앱 내의 모든 대화 기록, 설정, 시스템 프롬프트 등은 사용자의 기기(로컬 SQLite 데이터베이스)에 보관됩니다.

사용자는 언제든지 앱 내의 메뉴를 통해 채팅 방을 삭제하거나 폴더를 비울 수 있으며, 앱을 삭제(Uninstall)할 경우 기기에 저장된 모든 관련 데이터가 즉시 파기됩니다.

앱은 다음과 같은 기기 권한을 요청할 수 있으며, 각 권한은 명시된 목적 외로는 사용되지 않습니다.
